# 🎉 Event Management System

A web-based application developed to manage event registrations, scheduling, and administrative operations efficiently.

---

## 📌 Overview

The Event Management System is designed to simplify the process of organizing and managing events. It allows users to register for events, while administrators can manage schedules, participants, and event details through a centralized system.

---

## ✨ Features

- ✅ User registration for events  
- ✅ Event scheduling and management  
- ✅ Admin panel for managing events and users  
- ✅ View and manage participant details  
- ✅ Organized database storage  

---

## 🛠️ Tech Stack

- **Language:** Java  
- **Frontend:** HTML, CSS
- **Backend:** Java  
- **Database:** MySQL  
- **Tools:** NetBeans
